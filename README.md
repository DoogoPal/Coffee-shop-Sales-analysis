# Coffee-shop-Sales-analysis
This project focuses on analyzing transactional data from a coffee shop franchise, Maven Roasters, to identify business patterns, trends, and opportunities. The goal was to transform raw sales data into a dynamic and interactive dashboard using Microsoft Excel
The dashboard helps business owners understand customer behavior, monitor sales performance, and make data‑driven decisions to improve operations and profitability.

Objectives

1) Clean and prepare raw transactional data for analysis
2) Analyze sales performance using Excel PivotTables
3) Create an interactive and dynamic dashboard
4) Identify trends in sales, products, locations, and customer purchasing patterns

Dataset Information

The dataset contains transactional data from January 2023 to June 2023 and includes the following fields:

1) Transaction ID
2) Transaction Date
3) Store Location
4) Product Category
5) Product Name
6) Quantity Sold
7) Unit Price
8) Total Sales
